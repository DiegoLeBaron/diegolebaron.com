# My Portfolio

This is my personal portfolio site—a place to share who I am, what I've built, and how to get in touch. I built it with **Vue 3** because I wanted a modern, reactive front end that was fun to work with and easy to keep updated.

You’ll find a dark theme, smooth scrolling, a few project highlights in a carousel, my work history (with expandable timelines), and a skills section. Everything is responsive so it works on desktop and mobile.

## Running it locally

If you’d like to run it on your machine:

```bash
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173) in your browser.

To build for production:

```bash
npm run build
```

The built files will be in the `dist/` folder.

## Tech I used

Vue 3, Vite, Tailwind CSS, and Lenis for smooth scrolling. The content (projects, work history, skills) lives in the `src/data/` folder so I can update it without digging through components.
